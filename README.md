# Carta

A CLI tool for LLM-augmented development workflows.

## Installation

```bash
pip install -e .
```

## Usage

```bash
carta discover
```

### Commands

- `discover` - Discover and document requirements for a feature. Builds a cache of stripped Python files for LLM context.

## Development

```bash
pip install -e ".[dev]"
```

## Requirements

- Python 3.11+
